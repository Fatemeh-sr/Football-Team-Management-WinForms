# Football Team Management System (WinForms)

*Overview*
---

This project is a simple Windows Forms application developed in C# for managing and viewing football team player information.

It is an educational project that demonstrates:

Role-based login

Basic CRUD operations

Database connectivity

Windows Forms user interface

----

**Database**

- Microsoft Access (MDB)

- Data operations are performed using SQL queries inside the C# code

- The database is stored locally

**User Roles**

- Admin: Can view, search, add, edit, and delete player information

- User: Can only view and search player information

**Built With**

- C#

- .NET WinForms

- Microsoft Access Database

- SQL Queries

- Visual Studio
